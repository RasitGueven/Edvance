# Edvance – Produktbeschreibung

## Was ist Edvance?
Hybride Lernakademie für Schüler:innen Klasse 5–13. Präsenzunterricht in Kleingruppen (max. 5), jede:r arbeitet individuell am Tablet nach KI-generiertem Lernpfad. Coach begleitet: strukturiert, interveniert, motiviert.

## Personas
- Schüler:in: Klasse 5–13, braucht Struktur + Motivation
- Elternteil: will Transparenz + messbaren Fortschritt
- Coach: braucht Überblick + effiziente Tools
- Admin: verwaltet Standort, Buchungen, Berichte

## Kern-Versprechen
Pädagogische Qualität + operative Effizienz + messbarer Lernerfolg

## Preisstruktur
Basic / Standard / Premium

## Fächer
Mathematik, Deutsch, Englisch (Klasse 5–13)
