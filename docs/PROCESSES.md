## Kernprozesse (Reihenfolge)
1. Lead kommt rein → Ashkan Erstkontakt
2. Interesse bestätigt → Onboarding-Termin vereinbaren
3. Onboarding → Schülerdaten erfassen, Ziele, Termin buchen
4. Diagnose-Session → Lernstandserfassung vor Ort
5. Diagnose-Ergebnis → Elternreport + Lernplan erstellen
6. Session-Betrieb → wöchentliche Sessions, Home-Quests
7. Monatlicher Report → Eltern-App, Fortschritt sichtbar
